# PaymentGateways
https://www.linkedin.com/learning/asp-dot-net-core-payment-gateways

## What's presented:
* Integrate Braintree and Stripe payment gateways
* Create a single payment
* Subscribe to a payment plan
* Retrive and display Stripe's:
   * Products 
   * Transactions
   * Balance
   * Charges
   * Refunds & Disputes
   * Customers
